# Fitness Assist - Фитнес-приложение

## Описание
Мобильное приложение для отслеживания фитнес-активности, водного баланса и питания.

## Функционал
1. **Тренировки** - таймер для отслеживания времени тренировки
2. **Водный баланс** - отслеживание выпитой воды с настраиваемой целью
3. **Питание** - подсчет калорий с быстрым добавлением продуктов
4. **Настройки** - установка целей и сброс данных

## Технологии
- Язык: Kotlin
- Минимальная SDK: 21
- Архитектура: Single Activity с Fragment
- Хранение данных: SharedPreferences
- Навигация: Navigation Component

## Установка
1. Распакуйте ZIP-архив и положите папку приложения в папку с проектами Android Studio
2. Откройте проект в Android Studio
3. Запустите на эмуляторе или реальном устройстве

## Структура проекта
- `MainActivity.kt` - основная активность
- `HomeFragment.kt` - главный экран
- `WorkoutFragment.kt` - экран тренировки
- `WaterFragment.kt` - экран водного баланса
- `NutritionFragment.kt` - экран питания
- `SettingsFragment.kt` - экран настроек
